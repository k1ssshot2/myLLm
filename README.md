### How to run it on your own machine

1. Start virtual machine

   ```
   python -m venv venv
   venv\Scripts\activate
   ```

2. Install the requirements

   ```
   pip install -r requirements.txt
   ```

3. Run the app

   ```
   streamlit run app.py
   ```
